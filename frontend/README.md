# Frontend — Office Management System

React 19 + TypeScript + Vite + Material UI SPA for the Office Management System.

See the repository root [README.md](../README.md) and [docs/architecture.md](../docs/architecture.md)
for system-wide setup, conventions, and module documentation.

## Scripts

- `pnpm --filter frontend dev` — run dev server
- `pnpm --filter frontend build` — production build
- `pnpm --filter frontend test` — unit tests
- `pnpm --filter frontend lint` — lint
