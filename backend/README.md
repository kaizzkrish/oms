# Backend — Office Management System API

NestJS + Prisma + PostgreSQL + Redis API for the Office Management System.

See the repository root [README.md](../README.md) and [docs/architecture.md](../docs/architecture.md)
for system-wide setup, conventions, and module documentation.

## Scripts

- `pnpm --filter backend start:dev` — run in watch mode
- `pnpm --filter backend build` — production build
- `pnpm --filter backend test` — unit tests
- `pnpm --filter backend test:e2e` — end-to-end tests
- `pnpm --filter backend lint` — lint
